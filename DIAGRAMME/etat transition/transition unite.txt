title  états : Unite (StatutUnite)

[*] --> LIBRE : UniteService.create()\n/ Unité créée avec statut LIBRE par défaut

' ═══════════════════ DÉFINITION DES ÉTATS ET ACTIONS INTERNES ═══════════════════

LIBRE : description / Unité disponible à la location. Apparaît dans /api/bail/unite_libre.

OCCUPEE : entry / Associée à un ContratBail ACTIF
OCCUPEE : description / Unité sous contrat de bail actif. Non disponible à la location.

MAINTENANCE : description / Unité temporairement indisponible pour travaux.

' ═══════════════════ TRANSITIONS ENTRE LES ÉTATS ═══════════════════

LIBRE --> OCCUPEE : ContratBailService.addContrat()\n/ unite.setStatut(StatutUnite.OCCUPEE)
OCCUPEE --> LIBRE : ContratBailService.resilierContrat()\n/ unite.setStatut(StatutUnite.LIBRE)

LIBRE --> MAINTENANCE : PATCH /api/unite/changer_statut?statut=MAINTENANCE\n[rôle PROPRIETAIRE]
MAINTENANCE --> LIBRE : PATCH /api/unite/changer_statut?statut=LIBRE\n[rôle PROPRIETAIRE]

OCCUPEE --> MAINTENANCE : PATCH /api/unite/changer_statut?statut=MAINTENANCE\n[rôle PROPRIETAIRE — après résiliation préalable]

' ═══════════════════ NOTES ET CONTRAINTES ═══════════════════
note as N1
  <b>Règle métier :</b>
  Une unité OCCUPEE ne peut pas être attribuée à un nouveau bail
  sans résiliation préalable du bail en cours.
end note
@enduml
